# 行为准则

## 我们的承诺

我们承诺让每个人都能在社区中获得免受骚扰的参与体验，不因年龄、体型、可见或不可见的残障、族裔、性征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教、性身份或性取向而受到区别对待。

## 我们的标准

有助于营造积极环境的行为包括：

- 保持尊重和体贴
- 使用欢迎、包容的语言
- 接受建设性反馈
- 关注对社区最有利的事情

不可接受的行为包括：

- 骚扰或歧视
- 恶意挑衅、侮辱或贬损性评论
- 未经许可公开他人的私人信息

## 执行

社区维护者负责解释可接受行为的标准，并可针对不可接受的行为采取适当措施。

如果发现辱骂、骚扰或其他不可接受的行为，可以向项目维护者报告。
